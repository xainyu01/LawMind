## 下一步开发任务

### 任务 1：RAGAS 评测（推荐优先）

**目标**：搭建自动化评测框架，量化 RAG 系统质量。

**步骤**：
1. 安装 ragas：`uv add ragas`
2. 创建 `scripts/eval_ragas.py`
3. 准备评测数据集（question + ground_truth + contexts）
4. 运行评测，输出指标：
   - Faithfulness（答案忠实度）
   - Answer Relevancy（答案相关性）
   - Context Precision（上下文精确度）
   - Context Recall（上下文召回率）

**参考文档**：`文档/法律RAG系统.md` 第六节

### 任务 2：文件上传功能

**目标**：Streamlit 界面支持上传文档并入库。

**步骤**：
1. `app.py` 添加 `st.file_uploader`
2. 上传后调用 `document_loader` 解析
3. 调用 `ingest.py` 逻辑入库
4. 显示入库成功/失败状态

### 任务 3：用户反馈收集

**目标**：记录用户对回答的满意度。

**步骤**：
1. `app.py` 添加点赞/点踩按钮
2. 反馈存入 `data/feedback.jsonl`
3. 后续可用于评测数据集构建

## 注意事项

1. **GPU 操作需确认**：模型加载、训练等 GPU 操作前需二次确认
2. **HF_ENDPOINT 必须设置**：加载嵌入/重排序模型时必须有 `HF_ENDPOINT=https://hf-mirror.com`
3. **PYTHONPATH**：脚本运行时需要 `PYTHONPATH=.` 从 `app/` 导入模块
4. **PyTorch CUDA 索引**：`pyproject.toml` 中 `[tool.uv.sources]` 配置不能删除，否则会安装 CPU 版本
