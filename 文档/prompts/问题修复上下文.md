# 问题修复上下文提示词

> 使用方法：将此提示词与问题文档一起发送给 Claude，作为修复任务的上下文。

---

## 提示词模板

```
我遇到了以下问题需要修复。请先阅读问题文档，理解所有问题的背景和当前状态。

**问题文档**：[粘贴文档路径或内容]

**修复要求**：

1. **先理解后行动**：阅读完问题文档后，先向我确认你对问题的理解是否正确
2. **核对修复方案**：在执行修复前，必须与我核对以下信息：
   - 你打算使用哪个修复方案（如果有多个方案）
   - 修复步骤和预计时间
   - 可能的风险和回退方案
   - 需要我提供的信息或权限
3. **逐步执行**：修复过程中，每完成一个关键步骤，向我报告进度
4. **记录日志**：将修复过程中的关键信息（命令输出、错误信息、修复结果）记录到问题文档中
5. **验证修复**：修复完成后，执行验证步骤确认问题已解决

**注意事项**：
- 涉及 GPU 的操作需要二次确认（模型加载、训练等）
- 不要删除或覆盖重要数据，除非我明确同意
- 如果修复过程中遇到新问题，立即暂停并报告
```
