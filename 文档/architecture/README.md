## 核心架构

```
用户交互层（Streamlit / FastAPI）
        │
核心 RAG 引擎
├── 意图识别（LLM Prompt）
├── 混合检索（BM25 + 向量 + RRF）
├── Cross-Encoder 重排序
├── MMR 去重
└── LLM 生成（DeepSeek API）
        │
数据存储层
├── ChromaDB（向量存储）
├── Redis（缓存/记忆）
├── MySQL（用户/认证）
└── DiskCache（本地缓存）
```
