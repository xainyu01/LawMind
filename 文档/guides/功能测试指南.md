## 七、常见问题排查

### 问题 1：检索返回空结果

**可能原因**：
- ChromaDB 数据未加载
- 嵌入模型未正确初始化

**解决方法**：
```bash
# 检查数据
PYTHONPATH=. uv run python -c "from app.rag.vector_store import get_collection_count; print(get_collection_count())"

# 重新入库
HF_ENDPOINT=https://hf-mirror.com PYTHONPATH=. uv run python scripts/ingest_auto.py --split-mode legal data/datasets/scl/texts/
```

---

### 问题 2：混合检索超时

**可能原因**：
- BM25 索引需要重建
- 数据量过大

**解决方法**：
```bash
# 清除 BM25 缓存
rm -rf data/chroma/bm25_cache/

# 重新查询（会自动重建索引）
HF_ENDPOINT=https://hf-mirror.com PYTHONPATH=. uv run python scripts/search_test.py --hybrid "测试查询"
```

---

### 问题 3：where 过滤失败

**可能原因**：
- HNSW 索引损坏
- 元数据缺失

**解决方法**：

## 八、自动化测试脚本

创建 `scripts/test_all.py`：

```python
"""
自动化测试脚本

用法：
    uv run python scripts/test_all.py
"""

import sys
import time
from app.rag.vector_store import get_collection_count, search
from app.rag.retriever import get_retriever


def test_collection_count():
    """测试数据统计"""
    count = get_collection_count()
    assert count > 1000000, f"数据量不足: {count}"
    print(f"✅ 数据统计: {count} 条")


def test_vector_search():
    """测试向量检索"""
    results = search("夫妻义务", top_k=5)
    assert len(results) > 0, "向量检索返回空"
    print(f"✅ 向量检索: 返回 {len(results)} 条")


def test_hybrid_search():
    """测试混合检索"""
    retriever = get_retriever()
    results = retriever.search("夫妻义务", top_k=5)
    assert len(results) > 0, "混合检索返回空"
    print(f"✅ 混合检索: 返回 {len(results)} 条")


def test_search_performance():
    """测试检索性能"""
    start = time.time()
    search("夫妻义务", top_k=5)
    elapsed = time.time() - start
    assert elapsed < 1, f"检索超时: {elapsed:.3f}s"
    print(f"✅ 检索性能: {elapsed:.3f}s")


def main():
    print("=" * 60)
    print("功能测试")
    print("=" * 60)

    tests = [
        test_collection_count,
        test_vector_search,
        test_hybrid_search,
        test_search_performance,
    ]

    passed = 0
    failed = 0

    for test in tests:
        try:
            test()
            passed += 1
        except Exception as e:
            print(f"❌ {test.__name__}: {e}")
            failed += 1

    print("=" * 60)
    print(f"测试完成: {passed} 通过, {failed} 失败")
    print("=" * 60)

    return 0 if failed == 0 else 1


if __name__ == "__main__":
    sys.exit(main())
```
