## 下一步开发任务

### ~~任务 1：RAGAS 评测~~ ✅ 已完成

**目标**：搭建自动化评测框架，量化 RAG 系统质量。

**实现方案**：由于 `ragas` 库与 `langchain-community`（已 sunset）存在依赖冲突，采用自定义实现 RAGAS 风格评测，使用 DeepSeek 作为评判 LLM。

**文件**：
- `scripts/eval_ragas.py` — 评测脚本
- `data/eval/legal_eval_dataset.json` — 评测数据集（10 道法律题目）
- `data/eval/eval_results.json` — 评测结果

**评测指标**：
- Faithfulness（答案忠实度）：答案中的声明是否被上下文支持
- Answer Relevancy（答案相关性）：答案与问题的相关程度
- Context Precision（上下文精确度）：检索上下文中相关比例
- Context Recall（上下文召回率）：标准答案被上下文覆盖比例

**运行命令**：
```bash
HF_ENDPOINT=https://hf-mirror.com PYTHONPATH=. uv run python scripts/eval_ragas.py
HF_ENDPOINT=https://hf-mirror.com PYTHONPATH=. uv run python scripts/eval_ragas.py --verbose
```

**目标**：Streamlit 界面支持上传文档并入库。

## 注意事项

1. **GPU 操作需确认**：模型加载、训练等 GPU 操作前需二次确认
2. **HF_ENDPOINT 必须设置**：加载嵌入/重排序模型时必须有 `HF_ENDPOINT=https://hf-mirror.com`
3. **PYTHONPATH**：脚本运行时需要 `PYTHONPATH=.` 从 `app/` 导入模块
4. **PyTorch CUDA 索引**：`pyproject.toml` 中 `[tool.uv.sources]` 配置不能删除，否则会安装 CPU 版本

## Phase 10：用户认证与 MySQL ✅

### ~~任务 18：用户认证系统~~ ✅ 已完成

**目标**：完整的用户登录体系，Access + Refresh 双令牌，Streamlit 登录界面。

**实现**：
- `app/db/mysql_client.py` — SQLAlchemy 引擎、会话、建表
- `app/db/models.py` — User + RefreshToken ORM 模型
- `app/core/auth.py` — 双令牌认证（Access 60分钟 + Refresh 7天）、bcrypt 密码
- `app/api/auth_routes.py` — 注册/登录/刷新/登出/用户管理接口
- `app.py` — Streamlit 登录/注册页面、自动刷新、管理员面板
- `scripts/init_db.py` — 建库建表、创建默认管理员

**MySQL 表结构**：
```sql
CREATE TABLE users (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(128) NOT NULL,
    role VARCHAR(10) DEFAULT 'user',
    is_active BOOLEAN DEFAULT TRUE,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

CREATE TABLE refresh_tokens (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    user_id BIGINT NOT NULL,
    token_jti VARCHAR(36) UNIQUE NOT NULL,
    expires_at DATETIME NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

**API 接口**：
```bash
# 注册
curl -X POST http://localhost:8000/auth/register -H "Content-Type: application/json" -d '{"username":"test","password":"test123"}'

# 登录
curl -X POST http://localhost:8000/auth/login -H "Content-Type: application/json" -d '{"username":"admin","password":"admin123"}'

# 刷新 token
curl -X POST http://localhost:8000/auth/refresh -H "Content-Type: application/json" -d '{"refresh_token":"..."}'

# 获取当前用户
curl http://localhost:8000/auth/me -H "Authorization: Bearer <access_token>"

# 管理员查看用户列表
curl http://localhost:8000/auth/users -H "Authorization: Bearer <admin_token>"
```

**配置（.env）**：
```bash

JWT_ACCESS_SECRET=your-access-secret
JWT_REFRESH_SECRET=your-refresh-secret
ADMIN_USERNAME=admin
```
