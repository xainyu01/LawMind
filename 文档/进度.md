### 解决的关键问题

| 问题 | 方案 |
|------|------|
| numpy MINGW 崩溃 (Python 3.14) | 升级依赖链 → numpy 2.4.6 |
| huggingface 被墙 | `.env` 设 `HF_ENDPOINT=https://hf-mirror.com` |
| PyTorch CPU 版反复覆盖 | `pyproject.toml` 加 `[tool.uv.sources]` 锁定 CUDA 索引 |
| langchain 1.x `Document` 路径变更 | 改用 `langchain_core.documents` |

### 硬件确认

```
GPU:        NVIDIA GeForce RTX 5060 (8GB)
```
